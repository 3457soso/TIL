1. set(name, value) : 서버 설정을 위한 속성 설정. set() 메소드로 지정한 속성은 get()으로 꺼내 확인할 수 있다.
  - 속성의 종류
    A. env : 서버 모드를 설정함.
       a. PORT : 웹 서버 포트 설정.
    B. views : 뷰들이 있는 폴더 또는 폴더 배열 설정.
    C. view engine : 디폴트로 설정할 뷰 엔진 설정. ejs나 pug를 많이 사용한다고 함...
       뷰 엔진은 클라이언트에 보낼 응답 웹 문서를 만들 때 사용된다고 함.
       미리 템플릿을 만들어 놓고 그 템플릿을 사용해서 응답 웹 문서 생성...

2. get(name) : 서버 설정을 위해 지정한 속성을 꺼내옴.
  Ex) app.get('port') : port 설정한 값 가져오기.

3. use([path], function, [function]]) : 미들웨어 함수를 사용한다. (나중에 더 추가.)

4. get([path], function) : 특정 패스로 요청된 정보를 처리함.
