* 구글 맵 : 웹 페이지에 지도를 넣을 수 있는 방법은 여러 가지가 있지마 여기서는 구글맵은 사용한다!
  - 구글 맵을 이용하기 위해서는 API Key를 발급받아야 한다.

  - onLoad () : onLoad() 함수는 문서가 모두 로딩된 후에 호출된다.
  - initMap() : onLoad() 함수 내에서 호출된다. onLoad() 함수 내부에서 호출된다. 파라미터로는 2개가 주어진다.
    > 구글 맵은 google.maps.Maps 프로토타입으로부터 객체를 만들어낼 수 있음.
    > new 연산자와 함께 새로운 구글맵 객체가 만들어진다. 파라미터는 다음과 같다.
      1. 웹 페이지에 들어 있는 문서 요소.
      2. 지도를 초기화할 때 필요한 파리미터. 여기서는 center와 zoom 속성을 사용한다.
         center : 지도가 어느 지역을 보여줄 것인지를 좌표로 지정. js 객체에 lat와 lng 속성으로 좌표 값을 넣어줌.
         zoom : 지도 확대 수준 지정. 값이 클수록 자세한 지도 값을 받아옴.

  - marker : 내 위치도 아이콘으로 함께 보여 주기 위해서 사용
    > google.maps.Marker 프로토타입을 이용해 객체를 만들어 추가함.
    > 속성 중 animation을 지정해 움직이는 효과를 줄 수도 있다.
    > setMap 메소드를 호출해 지도 객체를 파라미터로 전달한다.

  - LatLngBounds({lat: ??, lng: ??}, {lat: ??, lng: ??}) : 사각형 영역을 표기하기 위한 좌표 설정
    > google.maps.Rectangle 프로토타입으로 사각형 객체를 만들어 지도에 추가!
