* 일대일 채팅에서는 상대방을 지정해서 메시지를 보내야 함.
  -> socket.io 모듈은 접속한 클라이언트마다 각자 고유한 아이디를 만들어 지정해줌.
  -> 하지만 이 아이디는 사람이 분간하기엔 복잡하다.
  -> 클라이언트가 로그인할 때 사용하는 로그인 아이디를 사용해서 클라이언트 구분!

  - io.sockets.connected 객체에 들어 있는 소켓 객체들 중에서 대상이 되는 소켓 객체를
    소켓 ID로 찾은 후, emit() 메소드를 호출해 메시지를 전송함!
    + sockets.connected 객체에 들어 있는 소켓들은 제각각 소켓 아이디를 가지고 있음!

* 그룹 채팅을 구현하기 위해서는 방이 필요함.
  - 먼저 방을 만들고 삭제하는 기능부터 추가한 다음에 메시지를 주고 받는 기능을 추가해야 함.
  - 방에 입장하거나 방에서 나갈 때는 socket.io 모듈이 지원하는 기능을 갖다 쓰면 된다!
    socket.join()  : 방에 입장할 때 호출
    socket.leave() : 방을 떠날 때 호출

  - io.sockets.in() 메소드를 이용해 대상이 되는 방의 소켓 객체를 찾은 후 메시지 전송.
