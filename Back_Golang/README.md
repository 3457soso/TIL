- **bin** : 바이너리 파일들이 들어간다. (실행 파일)
- **pkg** : 패키지 오브젝트 파일들이 들어간다.
  - 소스가 컴파일된 후의 코드들이 여기 들어감. but 실행 가능한 것 아님!
  - 라이브러리들이 들어간다고 생각하면 된다고 함.
- **src** : 소스 코드들이 들어간다. 여기에 내 패키지 경로를 작성하면 됨!
